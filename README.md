# 微信小程序支付 Java SDK
本项目主要分享在自己公司对接微信小程序支付的踩坑经历，以便后续再次用到的时候能够快速捡起来，同时也分享给各位小伙伴，因为微信官方提供的sdk有点坑，里面有很多缺陷，希望各位小伙伴能够参考此项目
跳出巨坑。
